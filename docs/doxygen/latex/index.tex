This documentation covers the STM32-\/based firmware used to drive the Avalon supercapacitor power manager board developed for RMUC 2025. The system manages high-\/speed power delivery, bidirectional energy control, and safety management for supercap-\/assisted robotics.\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_features_sec}

\begin{DoxyItemize}
\item Real-\/time power feedback control
\item CAN communication with robot control system
\item PID-\/based charge/discharge regulation
\item Voltage and current monitoring
\item Thermal supervision
