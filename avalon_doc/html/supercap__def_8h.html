Detailed Description

Configuration constants for the supercapacitor power manager firmware.

This file defines all global parameters for PID tuning, safety limits, thermal calculations, and voltage/current thresholds used throughout the supercapacitor controller logic.

Created on: Jul 5, 2024 Author: Yassine Bakkali
